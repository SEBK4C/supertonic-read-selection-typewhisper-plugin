# Notice

This repository contains a standalone TypeWhisper plugin derived from the
`TypeWhisperPluginSDK/Plugins/SupertonicPlugin` source in
`SEBK4C/typewhisper-mac`.

The plugin has been modified to use its own plugin identifier and principal
class, expose a direct "Read Selection" keyboard shortcut, and install without
colliding with TypeWhisper's first-party Supertonic marketplace plugin.

Supertonic model assets are not included in this repository or release bundle.
The plugin downloads them only after the user accepts the Supertonic 3 model
license in the plugin settings.
