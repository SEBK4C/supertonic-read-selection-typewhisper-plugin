# Supertonic Read Selection v1.1.0

This release focuses on faster local read-aloud startup and better long-selection throughput on Apple silicon Macs.

## What's New

- Added an optional Mac GPU backend using ONNX Runtime's Core ML execution provider with CPU and GPU compute units.
- Added sentence-aware chunk batching for long selections. Later chunks run in batches to reduce repeated ONNX session overhead.
- Added streaming playback. The plugin now schedules synthesized audio chunks as they finish instead of waiting for the entire selection.
- Added first-utterance priority. The first sentence, or the first short word-bound segment of a long sentence, runs as a one-item batch so speech can start sooner.
- Added runtime diagnostics for backend selection, chunk counts, first-batch size, emitted audio chunks, and synthesis timing.

## How It Works

When Use Mac GPU (Core ML) is enabled, the plugin asks ONNX Runtime to create the Supertonic sessions with the Core ML execution provider. The provider is configured for CPU and GPU compute units and uses a model cache directory next to the downloaded Supertonic assets. If Core ML cannot initialize the sessions, the plugin logs the failure and falls back to CPU.

Text is split into paragraphs, then sentence-aware chunks with abbreviation handling, and finally word-bound chunks for very long sentences. In streaming mode, the first utterance is promoted into its own small batch. After that first audio chunk is emitted, the remaining chunks continue in batches of four.

Playback uses a single audio player node. Each emitted audio buffer is queued onto that one player, so chunks play in order. If synthesis falls behind, playback may pause briefly between chunks, but chunks do not overlap.

## Install

1. Download `SupertonicReadSelectionPlugin.zip` from the release assets.
2. Open TypeWhisper settings.
3. Go to Integrations and choose Install Plugin.
4. Select the downloaded ZIP.
5. Enable Supertonic Read Selection.
6. Open Settings, accept the Supertonic 3 model license, download the model assets, choose a voice, and record the Read Selection Shortcut.

## Notes

- Supertonic model assets are downloaded by the plugin after license acceptance and are not bundled in this release.
- macOS Accessibility permission is required for direct selection access and clipboard fallback.
- Use Console or `log stream` with subsystem `com.sebk4c.typewhisper.tts.supertonic-read-selection` to inspect backend and chunk timing.
