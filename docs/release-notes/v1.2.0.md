# Supertonic Read Selection v1.2.0

This release focuses on more controllable fast playback, safer long-read behavior, and optional Supertonic expression tags.

## What's New

- Added player-side playback speed from 1x to 4x, separate from model generation speed.
- Added Advanced Audio controls for pitch, time-stretch overlap, low-pass cleanup, resonance cut, clarity shelf, and sibilance-taming EQ.
- Collapsed the detailed filter controls under Advanced Audio to keep Settings easier to scan.
- Added optional Expression Tags settings for the confirmed Supertonic tags `<laugh>`, `<breath>`, and `<sigh>`.
- Added automatic conversions for sentence breaths, `(lol)` / `(haha)` style laughter shorthand, and ellipses-to-sigh.
- Switched streaming long-read synthesis to one chunk at a time after first-utterance priority, reducing pauses and preventing overlapping playback.
- Added a predicted-duration guard so invalid model output is rejected before it can allocate runaway audio buffers.

## How It Works

Text is split into paragraphs, then sentence-aware chunks with abbreviation handling, and finally word-bound chunks for very long sentences. In streaming mode, the first utterance is promoted into its own short chunk so audio can begin quickly. Remaining chunks are synthesized serially and queued onto one audio player node, so playback stays ordered and chunks do not overlap.

Playback speed now lives in the AVAudioEngine playback chain instead of changing Supertonic generation speed. The Advanced Audio controls are all plugin-local and apply only to this plugin's playback session.

Automatic expression tags are opt-in. Official upstream docs currently confirm `<laugh>`, `<breath>`, and `<sigh>` by name; this plugin only auto-inserts those confirmed tags.

## Install

1. Download `SupertonicReadSelectionPlugin.zip` from the release assets.
2. Open TypeWhisper settings.
3. Go to Integrations and choose Install Plugin.
4. Select the downloaded ZIP.
5. Enable Supertonic Read Selection.
6. Open Settings, accept the Supertonic 3 model license, download the model assets, choose a voice, and record the Read Selection Shortcut.

## Notes

- Supertonic model assets are downloaded by the plugin after license acceptance and are not bundled in this release.
- macOS Accessibility permission is required for direct selection access and clipboard fallback.
- Use Console or `log stream` with subsystem `com.sebk4c.typewhisper.tts.supertonic-read-selection` to inspect backend, chunk timing, playback scheduling, and invalid-duration rejections.
